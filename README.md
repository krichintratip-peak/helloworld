# Hello World

My first GitHub project - a simple Python program that prints "Hello, World!" with a beautiful ASCII heart art.

## About

This is my first project on GitHub. It demonstrates a basic Python script that combines the classic "Hello, World!" message with ASCII art.

## Features

- Prints the classic "Hello, World!" message
- Displays a beautiful ASCII heart art

## How to Run

```bash
python hello.py
```
